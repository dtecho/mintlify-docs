## Authentication

All API endpoints require authentication using Bearer tokens. This security requirement is defined in our OpenAPI specification.

<Info>
  Authentication tokens should be included in the header of each request using the Bearer scheme.
</Info>

```json Code Example
{
  "security": [
    {
      "bearerAuth": []
    }
  ]
}
```

### Token Management

Keep your authentication tokens secure and never share them in public repositories or client-side code.

<Tip>
  Rotate your authentication tokens regularly and use environment variables to manage them in your applications.
</Tip>
