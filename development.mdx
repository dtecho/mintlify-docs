## Installation

<Info>
  **Prerequisites**: 
  - Node.js (version 19 or higher)
  - Updated ```docs.json``` file (legacy ```mint.json``` should be deleted)
</Info>

<Steps>
  <Step title="Install Mintlify CLI">
    Choose your preferred package manager to install Mintlify globally:

    <CodeGroup>
      ```bash npm
      npm i -g mintlify
      ```

      ```bash yarn
      yarn global add mintlify
      ```
    </CodeGroup>
  </Step>

  <Step title="Start Local Development">
    Navigate to your docs directory (where `docs.json` is located) and run:

    ```bash
    mintlify dev
    ```

    Your documentation will be available at `http://localhost:3000`.
  </Step>
</Steps>

## Local Preview

### Port Configuration

Mintlify runs on port 3000 by default. You can specify a different port using the `--port` flag:

```bash
mintlify dev --port 3333
```

<Note>
  If your specified port is already in use, Mintlify will automatically try the next available port:
  ```md
  Port 3000 is already in use. Trying 3001 instead.
  ```
</Note>

## Advanced Configuration

### Version Management

Each CLI release corresponds to a specific Mintlify version. To ensure your local preview matches production, keep your CLI updated:

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

## Development Tools

### Link Validation

Check for broken references in your documentation using the built-in validator:

```bash
mintlify broken-links
```

### Code Formatting

For the best development experience, we recommend:

1. [MDX VSCode extension](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) for syntax highlighting
2. [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) for code formatting

## Troubleshooting Guide

<AccordionGroup>
  <Accordion title='Sharp Module Error (darwin-arm64 runtime)'>
    Follow these steps to resolve the sharp module error:

    1. Remove existing mintlify: `npm remove -g mintlify`
    2. Upgrade to Node v19 or higher
    3. Reinstall mintlify: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Unknown Error Resolution">
    If you encounter an unknown error:
    1. Navigate to your root directory
    2. Delete the ~/.mintlify folder
    3. Run `mintlify dev` again
  </Accordion>
</AccordionGroup>
